### 개요

- Stock 대시보드 고도화의 정의: 단순 대시보드가 아닌 사용자와 interactive. 정보의 구조화
- 주요 구조적 변화
    - Interactive: Chatbot 탑재, 그래프 Customizing (대화 내용 실시간 반영)
    - 정보 구조화: 숫자/테이블형 데이터의 벡터화는 어떻게? 또는 그래프화 필요?
    - 메모리 탑재: Auth 적용 User의 경우 History DB 저장후 소환
- 주요 Add-on (or Agent)
    - Auth
    - Chatbot
    - 숫자/테이블 데이터 Vector DB
    - 그래프 Custom
    - Memory (by User RLS)
    - Onboard Agent
